\begin{wallpaperframe}
For every set \(S\), the power set \(\mathcal{P}(S)\) has strictly greater cardinality:
\[
|S| < |\mathcal{P}(S)|
\]
No surjection
\[
f : S \twoheadrightarrow \mathcal{P}(S)
\]
exists. The diagonal set \(D = \{\, x \in S \mid x \notin f(x) \,\}\) has no preimage.
\end{wallpaperframe}
