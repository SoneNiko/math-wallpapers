\begin{wallpaperframe}
The Mandelbrot set $\mathcal{M}$ is the set of complex numbers $c$ for which the sequence
\[
z_0 = 0, \qquad z_{n+1} = z_n^2 + c
\]
remains bounded, i.e.
\[
\mathcal{M} = \left\{ c \in \mathbb{C} \;\middle|\; \limsup_{n \to \infty} |z_n| \leq 2 \right\}
\]
\end{wallpaperframe}
