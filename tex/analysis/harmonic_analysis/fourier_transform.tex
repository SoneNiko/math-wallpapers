For an integrable function $f : \mathbb{R} \to \mathbb{C}$, its Fourier transform $\hat{f}$ is given by
\[
\hat{f}(\xi) = \int_{-\infty}^{\infty} f(x)\, e^{-2\pi i x \xi}\, dx
\]
with the inverse
\[
f(x) = \int_{-\infty}^{\infty} \hat{f}(\xi)\, e^{2\pi i x \xi}\, d\xi
\]
\end{wallpaperframe}
