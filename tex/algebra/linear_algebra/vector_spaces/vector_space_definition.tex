A vector space over a field $F$ is a set $V$ with operations $+$ and scalar multiplication, such that for all $u, v, w \in V$ and $a, b \in F$:
\begin{itemize}
  \item $u + v = v + u$
  \item $(u + v) + w = u + (v + w)$
  \item There exists $0 \in V$ with $u + 0 = u$
  \item For each $u$, there exists $-u$ with $u + (-u) = 0$
  \item $a(u + v) = au + av$
  \item $(a + b)u = au + bu$
  \item $a(bu) = (ab)u$
  \item $1u = u$
\end{itemize}
